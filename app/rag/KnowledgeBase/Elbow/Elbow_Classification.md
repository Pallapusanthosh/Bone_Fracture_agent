## Classification of Elbow Fractures

Elbow fractures are classified based on location and severity.

---

### 1. Supracondylar Fracture (Most Common)

Occurs above elbow (humerus)

#### Gartland Classification:

- Type I → Non-displaced
- Type II → Displaced (partial)
- Type III → Completely displaced
- Type IV → Unstable in all directions

---

### 2. Lateral Condyle Fracture

- Involves outer elbow
- Common in children

---

### 3. Medial Epicondyle Fracture

- Inner elbow injury
- Common in sports

---

### 4. Radial Head Fracture

#### Mason Classification:

- Type I → Minimal displacement
- Type II → Displaced
- Type III → Comminuted
- Type IV → With dislocation

---

### 5. Olecranon Fracture

- Tip of elbow (ulna)
- Intra-articular fracture

---

### 6. Coronoid Fracture

#### Regan & Morrey Classification:

- Type I → Tip
- Type II → <50%
- Type III → >50%

---

### 7. Monteggia Fracture

- Ulna fracture + radial head dislocation

---

### 8. Terrible Triad Injury

- Radial head fracture
- Coronoid fracture
- Elbow dislocation

---

### Key Insight

Displacement + neurovascular risk determines severity.
